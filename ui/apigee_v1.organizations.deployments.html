Apigee API . organizations . deployments

list(parent, sharedFlows=None, x__xgafv=None)

Lists all deployments of API proxies or shared flows.

list(parent, sharedFlows=None, x__xgafv=None) 
Lists all deployments of API proxies or shared flows.
  parent: string, Required. Name of the organization for which to return deployment information in the following format: `organizations/{org}` (required)
  sharedFlows: boolean, Optional. Flag that specifies whether to return shared flow or API proxy deployments. Set to `true` to return shared flow deployments; set to `false` to return API proxy deployments. Defaults to `false`.
