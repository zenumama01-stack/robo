Campaign Manager 360 API . floodlightActivities

"attributionEnabled": True or False, # Whether the activity is enabled for attribution. "status": "A String", # The status of the activity. This can only be set to ACTIVE or ARCHIVED_AND_DISABLED. The ARCHIVED status is no longer supported and cannot be set for Floodlight activities. The DISABLED_POLICY status indicates that a Floodlight activity is violating Google policy.
