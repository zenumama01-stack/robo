Pollen API . mapTypes . heatmapTiles

Returns a byte array containing the data of the tile PNG image.

Returns a byte array containing the data of the tile PNG image.
  mapType: string, Required. The type of the pollen heatmap. Defines the combination of pollen type and index that the map will graphically represent. (required)
      MAP_TYPE_UNSPECIFIED - Unspecified map type.
      TREE_UPI - The heatmap type will represent a tree index graphical map.
      GRASS_UPI - The heatmap type will represent a grass index graphical map.
      WEED_UPI - The heatmap type will represent a weed index graphical map.
  zoom: integer, Required. The map's zoom level. Defines how large or small the contents of a map appear in a map view. * Zoom level 0 is the entire world in a single tile. * Zoom level 1 is the entire world in 4 tiles. * Zoom level 2 is the entire world in 16 tiles. * Zoom level 16 is the entire world in 65,536 tiles. Allowed values: 0-16 (required)
