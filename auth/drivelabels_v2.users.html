getCapabilities(name, customer=None, x__xgafv=None) 
Gets the user capabilities.
  name: string, Required. The resource name of the user. Only "users/me/capabilities" is supported. (required)
  customer: string, The customer to scope this request to. For example: `customers/abcd1234`. If unset, it will return settings within the current customer.
    { # The capabilities of a user.
  "canAccessLabelManager": True or False, # Output only. Whether the user is allowed access to the label manager.
  "canAdministrateLabels": True or False, # Output only. Whether the user is an administrator for the shared labels feature.
  "canCreateAdminLabels": True or False, # Output only. Whether the user is allowed to create admin labels.
  "canCreateSharedLabels": True or False, # Output only. Whether the user is allowed to create shared labels.
  "name": "A String", # Output only. Resource name for the user capabilities.
